## Usage in Next.js

Reference these logos in your components:

```jsx
<img src="/partners/elevenlabs.png" alt="ElevenLabs" />
<img src="/partners/wiz.png" alt="Wiz.io" />
<img src="/partners/hubspot.png" alt="HubSpot" />
<img src="/partners/spline.png" alt="Spline" />
<img src="/partners/openrouter.png" alt="OpenRouter" />
```

All logos are PNG format with transparency support and optimized to ~180px width for logo sliders.
